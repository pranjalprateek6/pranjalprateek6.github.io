# pranjalprateek6.github.io

My personal portfolio — an interactive, terminal-style site shipped as a **single self-contained HTML file**. Fonts, imagery, the résumé PDF, and the app runtime are all inlined; the only runtime dependency is React/ReactDOM (loaded from a CDN with integrity hashes).

**Live:** https://pranjalprateek6.github.io/

## Local preview

No build step. Serve the folder with any static server and open it:

```sh
python -m http.server 8000
# then visit http://localhost:8000/
```
